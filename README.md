## 一些简单的自定义View集合

### VariInstructTextView

右侧带上升下降箭头的TextView，箭头右上角可增加数字。目前在宽度计算上有些小bug

### ChartView
曲线图，只有基础的功能

### CircleProgressView
包含两个view
CircleExponentView 类似速度表的样式
CircleProgressView 一个圆的进度圈
